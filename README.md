# Site coach sportif gratuit avec GitHub Pages + Pages CMS

## 1. Mettre en ligne

1. Crée un nouveau dépôt GitHub, par exemple `site-coach-papa`.
2. Envoie tous les fichiers de ce dossier dans le dépôt.
3. Va dans **Settings > Pages**.
4. Dans **Build and deployment**, choisis :
   - Source : **Deploy from a branch**
   - Branch : **main**
   - Folder : **/root**
5. Clique sur Save.
6. Le site sera disponible à une adresse du style :
   `https://ton-pseudo.github.io/site-coach-papa/`

## 2. Modifier sans coder

1. Va sur https://app.pagescms.org
2. Connecte-toi avec GitHub.
3. Installe l'application Pages CMS sur le dépôt.
4. Ouvre le dépôt dans Pages CMS.
5. Ton père pourra modifier :
   - Textes du site
   - Services
   - Vidéos YouTube
   - Articles / conseils
   - Galerie photos

## 3. Ajouter une vidéo

Dans Pages CMS :

1. Ouvre **Vidéos YouTube**.
2. Ajoute une ligne.
3. Mets un titre, une description et colle le lien YouTube.
4. Sauvegarde.

## 4. Ajouter une image

Dans Pages CMS :

1. Clique sur un champ image.
2. Téléverse l'image.
3. Sauvegarde.

## Important

Les vidéos ne doivent pas être uploadées dans GitHub. Il faut les mettre sur YouTube, puis coller le lien dans Pages CMS.
